# Bug Bounty Dorking Tool

A comprehensive tool for bug bounty hunters to generate and use Google dorks for finding vulnerabilities in web applications.

## Features

- Generate custom Google dorks for bug bounty targets
- Interactive web interface
- Educational lab platform with weekly modules
- API security testing capabilities

## Files

- `bug-bounty-dork-tool.html` - Main dork generation tool
- `README.md` - the process

## Usage

Open the HTML file in your web browser to start using the tools.
